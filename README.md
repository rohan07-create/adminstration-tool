# adminstration-tool
## Installation
Give Permission ```+x ``` to file using ```chmod ```

```bash
chmod +x adminstration-tool
```

## Options:
```bash
./adminstration-tool -f <file_name> [OPTION]

      -f :Users file
      -a :to add user with password
      -s :Get uptime of server through ssh
      -l :Stress alarm
      -d :User delete

```
